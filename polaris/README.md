<h2 align="center">How to Run</h2>

   ```
   - Clone this repository:
   $ git clone https://github.com/leclm/polaris.git

   or if you have an SSH key
   $ git clone git@github.com:leclm/polaris.git

   - Enter in directory:
   $ cd polaris/src

   - Run
   $ npm install
   $ ng serve

   - Open your browser in:
   http://localhost:4200/
   ```
